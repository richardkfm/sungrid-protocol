Makeover vs. new mechanic

Five tiers, from "renamed only" to "shipped its own C# trait." This is the answer to "which of these are just a fresh coat of paint" — read top to bottom, the roster gets progressively further from stock Red Alert.

76 · Stock reskin

Vanilla RA actor id, vanilla trait set, vanilla stats. Only the name and flavor text changed. Balance and the counters that already existed in Red Alert carry over unmodified — nothing to design here.

8 · New building

A genuinely new building with its own fantasy, assembled entirely from existing engine traits (GrantConditionOnPowerState, ProximityExternalCondition, CashTrickler…). No bespoke code, but new build-order choices that need their own counterplay — see the per-building "likely counters" already logged in docs/BUILDINGS.md.

2 · Weapon swap

ARCT and DISR replaced FTUR and E4 in the exact same tech-tree slot, same chassis art, same cost/HP/prerequisites — only the weapon changed (tone call, not a balance one). Counters are unchanged from what the units they replaced already had.

2 · New unit

SGDRO and SGDRS are genuinely new rosters entries with their own HP/cost/damage, one per faction. Both are Light-armored, so the existing shared E3 Rocket Soldier already counters either one at 100% bonus damage — a real counterpart already exists without new work.

1 · New C# logic

SILO (Battery Bank) is the only actor carrying genuinely new gameplay code — GridReserveVault, wired to GridReserveManager/GridReserveController for the Grid Reserve economic-victory mode. Everything else in the roster ships as data.

Parity flags

Specific asymmetries worth a look before calling the two sides "even," in order of how much they're actually Sungrid's doing versus inherited from Red Alert.

Drone Bay is a one-sided cost Worth deciding

SGDRN (Drone Bay, 900c) is Assembly-exclusive and is the only new building in the whole Phase 5 roster that isn't shared — every other new building (SGCRY, SGDAI, SGREL, SGSHL, SGSNS, SGTUR, RCYD) is buildable by both sides. The Consortium's equivalent unit, SGDRS, is produced from the Helipad it already has, so Assembly pays a whole extra structure to reach parity in drone access that Consortium gets for free off existing infrastructure.

Could be intentional — Assembly's "scavenged/improvised" identity fits needing dedicated new infrastructure where Consortium's "already-networked" identity doesn't — but it should be a deliberate call, not an accident of implementation order (see docs/BACKLOG.md issue #22–24's revision history for how SGDRN/SGDRO/SGDRS arrived at their current split).

Both drones already have a valid counterpart Already covered

SGDRO (350c / 3000 HP / 2×700 dmg) and SGDRS (600c / 3500 HP / 2×950 dmg) are both Light-armored. E3 Rocket Soldier — shared infantry, equal access on both sides, 100% damage vs. Light — already answers either one. No new counter unit is needed for these two; this part of the "does it have a valid counterpart" question is already resolved by an existing shared unit, not a gap.

Drone cost-efficiency isn't quite 1:1 Watch in playtest

SGDRS costs 71% more than SGDRO for +17% HP and +36% burst damage — worse raw cost-per-damage for the Consortium drone. That may be intentional compensation for not needing to build SGDRN first, but neither drone has been through a dedicated playtest pass yet (docs/BACKLOG.md issue #9 covers Grid Reserve balance specifically, not this pair) — flag it for the next balance session rather than assuming the numbers are final.

Consortium's static-defense tree is deeper Inherited, not introduced

Consortium has 6 Defense-queue entries (GAP, PDOX, AGUN, PBOX, HBOX, GUN) against Assembly's 4 (IRON, TSLA, ARCT, SAM) — Consortium gets two garrisonable pillbox variants and a dedicated turret where Assembly gets one Tesla Coil. This is stock Red Alert's own Allied/Soviet asymmetry, predating any Sungrid content, and the shared SGTUR (Grid Defense Turret) doesn't rebalance it since both sides can already build it. Worth knowing about even though it isn't something this project introduced.

Superweapons and the tech gate stay symmetric Confirmed even

PDOX/Chronosphere and IRON/Iron Curtain are a clean 1:1 pair, and both now share the same added sgdai prerequisite (Datacenter for AI) alongside their original tech-center gate — the Phase 5 revision that made SGDAI "actually gate something" (backlog issue #22) landed identically on both factions' superweapons plus the shared MSLO. No asymmetry introduced here.

Zombie / Giant Ant aren't standard roster content Just a note

Both require a bio prerequisite token that no building in the standard tech tree grants — it's only provided by a neutral bio actor placed on the Chernobyl map. They're map-specific bonus content, not part of either faction's normal build order, so they were excluded from the 89-actor headline count and the parity comparisons above.
