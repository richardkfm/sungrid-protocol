# Cameo source images

Drop full-resolution source renders here for the photographic build-menu
cameos (the "real-style" cameos that replace/augment the programmatic ones
generated by `mods/sungrid/bits/gen_concept_art.py`).

These are **source art only** — the shipped cameos are the cropped/fitted
`*icon.png` files under `mods/sungrid/bits/`. Keeping the originals here makes
the crops reproducible and documents provenance (see `docs/LICENSE_NOTES.md`).

## How to add one

1. Pull this branch locally.
2. Copy your image into this folder, e.g.:
   - `desert-base.png`  — the desert spec-sheet scene (covers most of the roster)
   - `grid-scene.png`   — the "NOVAYA ZARYA" hero scene (optional)
3. Commit and push. Claude will crop each subject onto the cameo panel
   (border + baked name label), preview at 4×, and wire the results into
   `mods/sungrid/bits/`.

## Guidance for good crops

- Higher resolution is better — a 64×48 cameo cut from a small region needs
  as much source detail as possible.
- One clearly readable subject per region; a simple/neutral backdrop behind
  each subject keeps the tiny crop from muddying.
